## 🎬 프로젝트 개요
인디픽터는 독립영화에 대한 접근성을 높이고자 만든 독립영화 전문 온/오프라인 영화 상영 사이트입니다.\
온라인 스트리밍, 오프라인 극장 예매, 채팅기능을 주요기능으로 제공합니다.
<br>
<br>

23.12.04 ~ 23.12.26 (총 23일)

- 펀딩 등록 승인, 펀딩 현황보기

- 온라인 상영기간/채팅방 오픈시간 설정, 오프라인 상영기간 설정
- 회원관리, 회원 삭제
- 회원 등급 변경, 초청권 발급
