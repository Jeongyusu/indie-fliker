## 🎬 프로젝트 개요
인디픽터는 독립영화에 대한 접근성을 높이고자 만든 독립영화 전문 온/오프라인 영화 상영 사이트입니다.\
온라인 스트리밍, 오프라인 극장 예매, 채팅기능을 주요기능으로 제공합니다.
